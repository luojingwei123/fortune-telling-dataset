# 中国算命术整体框架

## 一、传统分类：五术（山医命相卜）

道家把所有玄学系统归为"五术"，源自明澄派张耀文的分类，是目前最通用的骨架。

| 术 | 含义 | 与"算命"的关系 |
|---|---|---|
| **山** | 修仙、养生、静坐、丹道、符咒 | 不算命，是自我修炼 |
| **医** | 中医、方剂、针灸、灵疗 | 不算命，是治病 |
| **命** | 通过生辰八字等固定信息推算整体命运 | ⭐ 算命核心 |
| **相** | 通过外形（面、手、骨、宅、墓）观察 | ⭐ 算命核心 |
| **卜** | 通过起卦、随机现象预测具体事件成败 | ⭐ 算命核心 |

**算命 = 命 + 相 + 卜。**

- **命** 看的是"这一生的整体轨迹"（长期、宏观）
- **相** 看的是"当下的状态和禀赋"（静态、外显）
- **卜** 问的是"这一件具体的事会怎样"（短期、单点）

## 二、"命"类主要流派（推整体命运）

| 名称 | 起源/年代 | 输入 | 核心逻辑 |
|---|---|---|---|
| **四柱八字（子平术）** | 唐李虚中 → 宋徐子平 | 出生年月日时的干支（8 个字）| 用天干地支 + 五行生克推命 |
| **紫微斗数** | 相传宋陈抟 | 农历生辰 + 时辰 | 排 12 宫命盘，用 14 主星 + 辅佐煞星 |
| **五星推命（果老星宗）** | 唐张果老 | 出生时的星曜位置 | 类似西洋占星，中国古法 |
| **铁板神数** | 宋邵康节 → 清 | 生辰八字 + 六亲信息 | 查条文（"神数"），号称精确到父母兄弟 |
| **太乙神数** | 汉代 | 时间点 | 三式之一，主要推国运 |
| **河洛理数** | 宋 | 八字 | 用洛书 + 易理推算 |
| **九星气学** | 传入日本后独立发展 | 生年 | 用九宫 + 三元流年 |
| **姓名学** | 近代（日本传回） | 姓名笔画 | 五格数理 + 五行 |

**当代最流行的是"八字"和"紫微斗数"，其他流派多数从业者已很少。**

## 三、"相"类主要门类

| 名称 | 观察对象 |
|---|---|
| **面相** | 五官、气色、痣、纹路 |
| **手相** | 掌纹、指形、手色（中西两派都有） |
| **骨相** | 头骨、身骨形状（近代式微） |
| **摸骨** | 触诊骨骼判命 |
| **风水**（阳宅、阴宅） | 住宅、墓地环境 |
| **虹膜诊断学** | 西方引入 |

**面相、手相在网页产品化上门槛最低**（拍照识别 + 规则库匹配就能做出可玩性）。

## 四、"卜"类主要方法

| 名称 | 起卦方式 |
|---|---|
| **六爻（文王卦）** | 铜钱或摇卦，得六爻 → 六十四卦 |
| **梅花易数** | 时间、数字、事物起卦，宋邵康节创 |
| **奇门遁甲** | 时空排盘，"三式"之首，古称"帝王之学" |
| **六壬** | 排盘，三式之一 |
| **周易占卜** | 蓍草或钱币起卦 |
| **求签、抽签** | 寺庙签诗 |
| **塔罗牌** | 西方传入 |
| **梦占、测字** | 解梦、拆字 |
| **問杯（掷筊）** | 闽台常见 |

## 六、道家与算命的关系

道家本身**并没有一个统一的"道家算命术"**。所谓"道家算命"其实是指：

1. **五术里的"山"（修仙）由道家发展**，是道教的核心实践
2. **命卜相三术**在历史上很多由道士研究和传播（如陈抟、邵雍、徐子平都有道家背景）
3. **奇门遁甲、六壬、太乙**（三式）传统上被认为是"道门秘术"
4. **符咒、扶乩、降神**是道教直接操作的方式

**结论：** 主人想做"道家算命网页"的话，选题上比较自然的方向是：
- 八字（最经典、最有产品化空间）
- 紫微斗数（可视化命盘漂亮，网页表现力强）
- 奇门遁甲（"道家秘术"标签最强）
- 六爻/梅花易数（起卦交互有趣，占卜单事件）
- 面相/手相（拍照即用，门槛低）

## 七、科学界立场（写在前面免得踩坑）

- 一项 1958 年开始追踪 2000 多名同时出生英国人的研究：性格、职业、能力毫无相似性 → 否定占星理论
- 700 多名占卜师的研究：预测结果不优于随机猜测
- 心理学解释：**巴纳姆效应 + 冷读**（一般化描述 + 察言观色）
- 中华人民共和国刑法：算命本身不违法，但**收费高 + 出售"改运物"**容易被以"利用封建迷信虚构事实诈骗罪"定罪

**⚠️ 产品定位建议：文化科普 / 娱乐向 / 免费或象征性收费**，不要走"付费改运/大师亲算"路子。
