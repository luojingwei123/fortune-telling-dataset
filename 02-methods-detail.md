# 每种算命方法的核心解读方式

**用途：** 给主人做网页设计参考。每一项都写清楚：**输入什么 → 中间怎么处理 → 输出什么**，方便判断能不能程序化。

---

## 一、八字（子平术，最主流）

### 输入
- **出生年、月、日、时**（精确到时辰，2 小时一个）
- 出生地经纬度（讲究的会算真太阳时）

### 排盘
把出生时间转成 4 组"天干+地支"，共 8 个字：

```
      年柱   月柱   日柱   时柱
天干:  庚     丙     甲     戊
地支:  午     戌     子     辰
```

- 换算规则：以**立春**为一年开始，以**节气**为月分界
- 每个干支对应五行属性（金木水火土）

### 解读
1. **日干（日柱天干）= 命主本人**（例：日干"甲"，命主属木）
2. 其他 7 个字与日干的关系形成 **"十神"**：比肩、劫财、食神、伤官、正财、偏财、正官、七杀、正印、偏印
3. **五行强弱**：数 8 字里五行分布，判断"喜用神"（缺什么、忌什么）
4. **大运/流年**：以月柱起，每 10 年一步大运；配合每年流年再看
5. 结合刑冲合会、神煞（桃花、天乙贵人等）综合断

### 输出
- 性格倾向
- 事业、财运、婚姻、健康的走势
- 一生大致的高峰低谷（几岁行什么运）

### 网页可行性
⭐⭐⭐⭐⭐（最高）
- 排盘完全可算法化（有开源库 `lunar-javascript`、`sxtwl`）
- 解读部分：五行喜忌 + 十神组合 + 大运流年 → 可以规则库化
- 已有大量商业化产品参考

## 二、紫微斗数

### 输入
- 出生年、月、日、时辰（农历）
- 性别

### 排盘
1. 画一张**十二宫命盘**（正方形分 12 格）
2. 十二宫代表人生 12 个领域：**命宫、兄弟、夫妻、子女、财帛、疾厄、迁移、奴仆（交友）、官禄（事业）、田宅、福德、父母**
3. 根据出生时间计算 **紫微星** 在哪一宫，然后 14 主星（紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军）依规则排布
4. 加上辅星（左辅右弼、文昌文曲）、煞星（火铃、擎羊陀罗、地空地劫）、四化星（化禄、化权、化科、化忌）

### 解读
- **看某个宫位里有什么星** → 判断这个领域的状况
- 例：命宫有紫微 + 天府 = 领袖气质、稳重；命宫有破军 + 火星 = 冲动、多变
- **三方四正**：每个宫要看对宫、拱宫的星一起判断
- **大限/流年**：10 年一大限，每年一流年，看当年宫位

### 输出
- 命盘可视化（正方形 12 格图）
- 每个领域的详细描述

### 网页可行性
⭐⭐⭐⭐⭐
- 排盘算法明确（开源库 `iztro` 很成熟）
- **命盘可视化非常有网页表现力**（这是紫微斗数在网页上最大优势）
- 解读复杂但可以按主星组合建规则库

## 三、六爻（文王卦）

### 输入
- 一个具体问题（如"我这次面试能过吗"）
- 起卦方式（三选一）：
  - 三枚铜钱掷 6 次（网页上就是随机数）
  - 时间起卦
  - 报数字起卦

### 排盘
1. 每次掷钱得阴阳，6 次形成一个"本卦"（6 爻从下往上）
2. 有变爻的话形成"变卦"
3. 装上 **纳甲**（每爻配天干地支）、**六亲**（父母/兄弟/子孙/妻财/官鬼）、**六神**（青龙朱雀勾陈螣蛇白虎玄武）
4. 定"用神"（跟你问的事有关的六亲：问功名看官鬼，问财看妻财……）

### 解读
1. 看用神在哪一爻
2. 看用神旺衰（是否得日月生扶）
3. 看动爻对用神的影响（生、克、冲、合）
4. 综合断吉凶

### 输出
- 卦象图
- 直接给"能/不能"、"何时能"的判断

### 网页可行性
⭐⭐⭐⭐
- 起卦交互特别适合网页（点击"摇卦"，动画掷铜钱）
- 装卦完全算法化（`iztro` 也支持）
- 解读规则库需要功夫，但比八字/紫微简单

## 四、梅花易数

### 输入
- **任何一个数字、时间点、看到的事物**
- 例：现在时间是 15:20，就用时辰起卦

### 排盘
1. 用输入的数除以 8 取余 → 上卦
2. 加时辰再除以 8 取余 → 下卦
3. 上下两卦合成 6 爻 → 本卦
4. 再除以 6 取余 → 动爻位置 → 变卦

### 解读
- 看本卦、变卦、互卦
- 看动爻辞（《周易》原文）
- 结合"体用"关系判断吉凶

### 网页可行性
⭐⭐⭐⭐（很适合小工具）
- **交互极简：输入两个数字就出结果**
- 适合做成"今日一卦""每日运势"类小组件

## 五、奇门遁甲（"帝王之学"）

### 输入
- 一个时间点（求测时间）
- 一个求测方向（东南西北）
- 或一件具体事

### 排盘
- 以 **洛书九宫** 为盘（3×3 九格）
- 排入：**三奇**（乙丙丁）、**八门**（开休生伤杜景死惊）、**八神**（值符、腾蛇、太阴等）、**九星**
- 一个时辰一个盘，一年 4320 局
- 分"阴遁 / 阳遁"，冬至到夏至用阳遁

### 解读
- 看你想问的方向那一宫里有什么门、什么星、什么神
- 例：出行看"生门"位置，考试看"景门"，官非看"死门"
- **策略性极强**：不但告诉你吉凶，还告诉你去哪个方向、几点走

### 输出
- 一个 3×3 九宫盘
- 具体行动建议

### 网页可行性
⭐⭐⭐⭐（视觉冲击力强）
- 3×3 九宫盘可视化好看
- 排盘算法有开源实现
- **"道家秘术"标签最强**，营销点足

## 六、面相

### 输入
- 一张正面清晰的脸部照片

### 判读要点
1. **三停**：额头（上停，早年）、眉眼到鼻尖（中停，中年）、鼻底到下巴（下停，晚年）
2. **五官**：眉、眼、鼻、口、耳（各自代表不同信息，如鼻代表财、眼代表精神）
3. **十二宫**：命宫（印堂）、财帛（鼻）、兄弟（眉）、夫妻（眼尾）、子女（眼下）……
4. **气色**：脸部区域的气色变化预示近期运势

### 网页可行性
⭐⭐⭐⭐⭐（门槛最低但技术要求高）
- 需要用 **人脸关键点识别**（如 face-api.js、MediaPipe）
- 定位后按传统面相规则输出解读
- **注意：这块 AI 有明显偏差风险**，做娱乐向可以，别做严肃判断

## 七、手相

### 输入
- 一张手掌照片（左手或右手，男左女右传统）

### 判读要点
1. **三主线**：生命线（拇指下弧线）、智慧线、感情线
2. **辅助线**：命运线、太阳线、婚姻线、健康线
3. **丘**：金星丘、木星丘、土星丘、太阳丘、水星丘、月丘、火星丘
4. **指形、指长**、手色

### 网页可行性
⭐⭐⭐（技术门槛高）
- 需要手部关键点识别 + 手纹检测
- 直接用户上传图片 + 规则匹配可行但精度差
- 建议：让用户对照示意图自己勾几个特征，比 AI 识别更靠谱

## 八、姓名学（近代产物）

### 输入
- 姓名（中文汉字）

### 判读
1. 数**每个字的笔画**
2. 组合成 **五格**：天格、人格、地格、外格、总格
3. 每个数字（1-81）对应"数理吉凶"和五行属性
4. 综合五格看性格、事业、婚姻

### 网页可行性
⭐⭐⭐⭐⭐（超简单）
- 完全算法化
- 输入名字 3 秒出结果
- 适合做入口小工具

## 九、周易六十四卦查询

### 输入
- 一个卦名，或者起卦得到一个卦

### 输出
- 该卦的**卦辞、爻辞、彖传、象传**（《易经》原文）
- 现代白话解读

### 网页可行性
⭐⭐⭐⭐⭐（数据库题）
- 64 卦 × 6 爻 = 384 条，静态数据就能做
- 适合做"每日一卦"、"随机抽卦"

## 十、择日 / 黄历

### 输入
- 想要做的事（婚嫁、动土、开业、出行）
- 时间范围

### 输出
- 该范围内的宜/忌
- 冲煞信息

### 网页可行性
⭐⭐⭐⭐⭐
- 完全查表可算
- 开源库很多（`lunar-javascript`）
