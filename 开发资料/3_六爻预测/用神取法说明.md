# 用神取法说明

## 一、什么是用神？

**用神**是断六爻的**核心概念**。当求测某件事时，必须找到卦中的一个爻来**专门代表所占之事**，这个爻就叫"用神"。

**断卦流程**：确定用神 → 看用神的旺衰 → 结合动爻变化 → 得出结论。

**没有用神就无法断卦**。用神选错，全盘皆错。

## 二、按占事内容取用神

### 1. 占求财（钱、生意、投资）→ **妻财爻**

- 妻财爻旺相且生世 → 财运佳
- 妻财爻空亡（当旬无气）→ 求财困难
- 妻财爻被克 → 破财

### 2. 占官职、事业、丈夫（女）→ **官鬼爻**

- 女占丈夫：官鬼为夫
- 官鬼爻旺 → 事业顺、婚事成
- 官鬼被克或空 → 官职难就、无夫

### 3. 占父母、长辈、老师、房屋、契约、文书 → **父母爻**

- 学生占学业：父母爻代表文书成绩
- 占买房：父母爻代表房产
- 父母爻旺 → 事顺、有靠山

### 4. 占子女、宠物、下属、医药 → **子孙爻**

- 占孩子：子孙爻为子女
- 占病：子孙爻为医药、贵人
- 子孙爻旺相 → 子孙聪明、病可愈

### 5. 占兄弟、朋友、同事、竞争者、口舌是非 → **兄弟爻**

- 占合伙生意：兄弟爻代表合伙人
- 兄弟爻旺动 → 竞争激烈、口舌事多
- 兄弟爻克财爻 → 破财

### 6. 占自己（问自身）→ **世爻**

- 世爻旺相 → 自身情况良好
- 世爻被克 → 自身受阻

### 7. 占对方（他人）→ **应爻**

- 应爻旺相 → 对方情况好
- 应爻克世 → 对方对我不利

## 四、用神的判断要素

### 1. 旺衰

**旺相**（用神有力）：
- 得月令、日辰生扶
- 有动爻生助
- 与卦身相合

**衰弱**（用神无力）：
- 被月令、日辰克制
- 无生助或被冲破
- 逢空亡

### 2. 生克关系

- 用神被生 → 事情有推动力
- 用神被克 → 事情受阻
- 用神生他爻 → 泄气（不利）
- 用神克他爻 → 主动出击（有利）

### 3. 动静

- 用神发动（动爻）→ 事情动，有变化
- 用神安静 → 事情静，看当前状态
- 用神动化吉/凶 → 结果吉/凶

### 4. 空亡

**旬空**：当日干支的"旬"内缺失的两个地支为空亡。

**举例**：甲子日，甲子旬内十日的干支是甲子、乙丑、丙寅、丁卯、戊辰、己巳、庚午、辛未、壬申、癸酉，那么"戌"和"亥"就是旬空。

**空亡的意义**：
- 用神空亡 → 事无成、事拖延
- 忌神空亡 → 忌神无力（反而好事）
- 空亡逢冲 → 冲开空亡，事可动

### 5. 伏神/飞神

**伏神**：用神在卦中不显，藏在其他爻之下。

**如果没有用神显现，需从"本宫首卦"取伏神**。

例：占求财，卦中无妻财爻显现，则取本宫首卦相应位置的妻财爻为伏神。伏神与飞神（现在这爻）的生克关系判断吉凶。

## 五、原神、忌神、仇神

**原神**：生用神的爻（用神的"帮手"）

**忌神**：克用神的爻（用神的"敌人"）

**仇神**：克原神的爻（帮手的敌人，间接伤害用神）

**综合判断**：
- 原神旺 → 用神有依靠
- 忌神旺 → 用神受威胁
- 用神+原神都旺 → 大吉
- 用神+忌神都旺（相冲）→ 大凶

## 六、断卦步骤总结

1. **明确所求**：占的是什么事？
2. **取用神**：按事项确定用神爻
3. **看用神状态**：
   - 旺相还是衰弱？
   - 是动爻还是静爻？
   - 是否空亡？
   - 是否有伏神/飞神？
4. **分析生克**：
   - 谁生用神（原神）？
   - 谁克用神（忌神）？
   - 用神与世爻的关系？
5. **看变卦**：动爻变化后用神如何？
6. **综合结论**：吉？凶？何时应验？

## 七、断卦示例

**例**：某人问求财，得**火风鼎**（无动爻）

- 火风鼎属**离宫**，卦宫五行为**火**
- 六亲配置（离宫为"火"）：
  - 木 → 父母（生我火）
  - 火 → 兄弟（同我）
  - 土 → 子孙（我生）
  - 金 → 妻财（我克）
  - 水 → 官鬼（克我）
- 找**妻财爻**（用神）→ 找卦中五行为金的爻
- 火风鼎各爻纳甲：初爻辛丑土、二爻辛亥水、三爻辛酉金、四爻己酉金、五爻己未土、上爻己巳火
- 妻财在三爻（辛酉金）和四爻（己酉金）
- 世爻在二爻（辛亥水）
- 妻财爻旺相（当月是秋季金旺）→ 有财
- 妻财爻生世爻（金生水，但爻间关系）→ 用神与世无直接互动
- **结论**：财运平平，需主动求取
