# 六爻摇卦规则

## 什么是六爻？

六爻是**周易占卜**中最主流的方法，用三枚铜钱**投掷 6 次**（每次得到一爻），从下往上排列，形成一个 6 爻卦。配合**纳甲、六亲、世应、用神**等方法进行断卦。

## 一、准备工作

**工具**：
- 三枚相同的铜钱（或硬币）
- 一个平面（桌子、盘子）
- 一颗诚心

**古法**：三枚"乾隆通宝"或"康熙通宝"，字面为"阳"，花纹面为"阴"。

**现代替代**：三枚一元硬币，国徽面为"字"（阳），一元面为"背"（阴）。

## 二、投掷方法

### 步骤：

1. **静心**：将求测问题清晰地想在心里。
2. **摇卦**：手心捧铜钱，摇动数秒，掷向桌面。
3. **观察**：看三枚硬币中"字面"和"背面"的分布。
4. **记录**：这是**第1爻（初爻）**。
5. **重复6次**：共投6次，从下往上依次记录为**初爻、二爻、三爻、四爻、五爻、上爻**。

### 每爻的记录方式：

| 字面数 | 背面数 | 记号 | 爻性 | 是否动爻 |
|--------|--------|------|------|----------|
| 3 字 | 0 背 | 老阳 ○ | 阳 | **动**（阳变阴）|
| 2 字 | 1 背 | 少阳 ⚊ | 阳 | 静 |
| 1 字 | 2 背 | 少阴 ⚋ | 阴 | 静 |
| 0 字 | 3 背 | 老阴 × | 阴 | **动**（阴变阳）|

**关键**：
- 三个都同面 → **动爻**（变阴/变阳）
- 三个不同面 → **静爻**（不变）

**动爻是断卦的重点**，一次摇卦可能有 0-6 个动爻。

## 三、卦象生成

### 例：

摇卦6次结果（从初爻到上爻）：
- 初爻：2字1背 → 少阳 ⚊
- 二爻：3字 → 老阳 ○（动，变阴）
- 三爻：1字2背 → 少阴 ⚋
- 四爻：2字1背 → 少阳 ⚊
- 五爻：0字3背 → 老阴 ×（动，变阳）
- 上爻：1字2背 → 少阴 ⚋

**本卦**（六爻从下到上）：
```
上爻 ⚋   （少阴）
五爻 ×  （老阴，动）
四爻 ⚊
三爻 ⚋
二爻 ○  （老阳，动）
初爻 ⚊
```

- 下卦（初、二、三爻）：⚊○⚊ → 离 ☲
- 上卦（四、五、六爻）：⚊×⚋ → 巽 ☴（因为五爻记录时先看当前值）

本卦：**风火家人**

**变卦**：动爻反过来。
- 二爻由阳变阴：下卦变成 ⚊⚋⚊ → 离... 等等，需要重新计算。

（实际操作：动爻位置的阴阳翻转，得到新的爻，重新看六爻是什么卦。）

## 四、爻位的名称

从下往上：

| 位置 | 名称 | 象征（一般）|
|------|------|-------------|
| 第6爻（顶）| 上爻 | 事物的极点、老者、领导 |
| 第5爻 | 五爻 | 君主、领导、事情主体 |
| 第4爻 | 四爻 | 大臣、宰相、辅助 |
| 第3爻 | 三爻 | 危险位置、过渡阶段 |
| 第2爻 | 二爻 | 民、下属、稳定 |
| 第1爻（底）| 初爻 | 事情起点、初始状态 |

**上下卦分界**：一至三爻属下卦（内卦），四至六爻属上卦（外卦）。

## 五、动爻的意义

**动爻是断卦的关键**：
- 没有动爻 → 事情静止，看本卦爻辞
- 1 个动爻 → 主要动爻决定走向
- 2 个动爻 → 参考"两爻俱动，主客分明"，看阴动/阳动
- 3 个动爻 → 三爻皆动，动多不动
- 4 个及以上动爻 → 动多，看不动的爻
- 6 个动爻 → 六爻皆动，本卦爻辞不用，看变卦

**主动爻规则**（多爻动时）：
- 只看阴动或阳动最少的那个（例：4阳1阴动，看阴的）
- 或看某爻是"用神"位置

## 六、其他起卦方法

除了铜钱法，六爻也可以用：

1. **蓍草法（古法）**：用49根蓍草反复操作（较繁琐，现今很少用）
2. **数字法**：类似梅花易数，报数得卦
3. **金钱卦**（三枚铜钱）：**最常用**（就是上述方法）
4. **报卦法**：他人报告的第一句话或第一件事作为起卦依据

## 七、断卦前的必备信息

摇好卦后要标注：
1. **年月日时**（当日的天干地支）→ 用于纳甲
2. **本卦**（6爻）
3. **动爻位置**（可能多个）
4. **变卦**（把动爻翻转后的卦）
5. **世应位置**（每个卦固定，见"世应定位规则"）
6. **纳甲六亲**（每爻的干支和六亲关系，见"纳甲与六亲配置表"）

## 八、网页实现的六爻起卦

```javascript
// 模拟三枚铜钱
function tossCoins() {
    const results = [];
    for (let i = 0; i < 6; i++) {
        let yang = 0;
        for (let j = 0; j < 3; j++) {
            if (Math.random() > 0.5) yang++;
        }
        let yaoType;
        if (yang === 3) yaoType = 'old_yang';     // ○ 老阳，动
        else if (yang === 2) yaoType = 'young_yang'; // ⚊ 少阳
        else if (yang === 1) yaoType = 'young_yin';  // ⚋ 少阴
        else yaoType = 'old_yin';                   // × 老阴，动
        
        results.push(yaoType);
    }
    return results; // [初爻, 二爻, 三爻, 四爻, 五爻, 上爻]
}

// 输入：六个爻类型
// 输出：本卦 + 变卦 + 动爻位置 + 六亲配置
function generateGua(yaoList) {
    const benGua = yaoList.map(y => y === 'old_yang' || y === 'young_yang' ? 1 : 0);
    const dongYao = yaoList.map((y, i) => (y === 'old_yang' || y === 'old_yin') ? i : null).filter(x => x !== null);
    const bianGua = benGua.map((y, i) => dongYao.includes(i) ? (1 - y) : y);
    
    return { benGua, bianGua, dongYao };
}
```
