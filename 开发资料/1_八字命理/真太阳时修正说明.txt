# 真太阳时修正说明

## 什么是真太阳时？

八字排盘要求使用**出生地当地的真太阳时**，而非我们日常使用的"北京时间"（标准时间）。

**为什么？** 八字的时柱依据日晷正午（太阳在头顶最高点）来定，与地球自转位置强相关。北京时间是东八区（东经120°）的标准时间，对全国统一，但实际每个地方经度不同，太阳过顶时刻不同。

## 修正公式

真太阳时 = 北京时间 + 经度时差 + 均时差

### 1. 经度时差（最主要）

**公式：** 经度时差 = (当地经度 - 120) × 4 分钟

- 每偏离标准经线1度，时差4分钟
- 经度大于120°（东部）→ 时差为正（真太阳时比北京时间快）
- 经度小于120°（西部）→ 时差为负（真太阳时比北京时间慢）

**举例：**
- 北京（116.4°E）：(116.4-120)×4 = -14.4 分钟 → 真太阳时比北京时间慢约14分钟
- 上海（121.5°E）：(121.5-120)×4 = +6 分钟 → 快6分钟
- 乌鲁木齐（87.6°E）：(87.6-120)×4 = -129.6 分钟 → 慢约2小时10分钟
- 广州（113.3°E）：(113.3-120)×4 = -26.8 分钟 → 慢约27分钟

### 2. 均时差（可选，精细修正）

由于地球公转轨道是椭圆、地轴倾斜等原因，太阳并不总在正午12点（真太阳时）到达头顶，全年会有 -14 ~ +16 分钟的偏差。

**均时差参考表（每月中旬近似值）：**

## 特殊情况

### 夏令时（DST）修正

**中国大陆曾在 1986-1991 年实行夏令时**，如出生日在这些年份的4月中旬-9月中旬，北京时间比实际时钟快1小时，需减去1小时再修正经度时差。

具体时段：
- 1986: 5月4日 - 9月14日
- 1987: 4月12日 - 9月13日
- 1988: 4月10日 - 9月11日
- 1989: 4月16日 - 9月17日
- 1990: 4月15日 - 9月16日
- 1991: 4月14日 - 9月15日

### 子时的特殊处理

八字中"子时"跨越两天（23:00-次日01:00），存在两种流派：

- **早子时/夜子时派**：23:00-00:59 属于当日子时（日柱不变，时柱用甲子/丙子…）
- **子时换日派**：23:00 就换到下一天（日柱前进一天）

**主流做法**：多数派采用"子时换日"，即 23:00 已进入次日。

## 网页实现要点

1. **输入项**：需要用户输入出生地（可用城市列表或经度输入）
2. **修正流程**：
   - 用户输入北京时间 → 加/减夏令时（如适用） → 加经度时差 → 可选加均时差 → 得真太阳时
3. **可显示两个结果**：默认按真太阳时排，同时给出北京时间下的排盘对比
4. **公式伪代码**：
   ```
   真太阳时（分钟） = 北京时间（分钟） 
                    - (夏令时?60:0)
                    + (经度 - 120) × 4
                    + 均时差（月份对照表）
   ```
