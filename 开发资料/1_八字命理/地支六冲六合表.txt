# 地支冲合表

## 地支六冲（相冲，为凶）

对宫相冲，阴阳属性相同：

子 ⇌ 午（水火冲）
丑 ⇌ 未（土土冲，墓库冲）
寅 ⇌ 申（木金冲）
卯 ⇌ 酉（木金冲）
辰 ⇌ 戌（土土冲，墓库冲）
巳 ⇌ 亥（火水冲）

## 地支六合（相合，为吉）

子 + 丑 → 合化土
寅 + 亥 → 合化木
卯 + 戌 → 合化火
辰 + 酉 → 合化金
巳 + 申 → 合化水
午 + 未 → 合化火/土（无变化，实为日月合）

## 地支三合（力量最强的合）

三个地支形成一个五行的"生-旺-墓"三角：

申 + 子 + 辰 → 合化水局（申生子旺辰墓）
亥 + 卯 + 未 → 合化木局
寅 + 午 + 戌 → 合化火局
巳 + 酉 + 丑 → 合化金局

## 地支三会（同五行相聚，最强）

四季各三个月的地支相聚：

寅 + 卯 + 辰 → 会东方木局（春）
巳 + 午 + 未 → 会南方火局（夏）
申 + 酉 + 戌 → 会西方金局（秋）
亥 + 子 + 丑 → 会北方水局（冬）

## 地支相刑（不和，次凶）

无恩之刑：寅刑巳、巳刑申、申刑寅
恃势之刑：丑刑戌、戌刑未、未刑丑
无礼之刑：子刑卯、卯刑子
自刑：辰辰、午午、酉酉、亥亥

## 地支相害（相穿，为破）

子 未 相害
丑 午 相害
寅 巳 相害
卯 辰 相害
申 亥 相害
酉 戌 相害

## 地支相破

子 酉 相破
午 卯 相破
辰 丑 相破
戌 未 相破
寅 亥 相破
巳 申 相破

## 优先级判断

当同一个地支同时出现多种关系时，优先级：
1. 三会 > 三合 > 六合 > 六冲 > 六害 > 六破 > 相刑
2. 合冲同时出现：先看远近，近者力大
3. 合能解冲，冲能破合，视具体位置和数量而定
