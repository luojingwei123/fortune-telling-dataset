# 八字排盘规则

## 一、什么是八字？

八字（也称"四柱八字"、"生辰八字"）由一个人出生的**年、月、日、时**分别用一对天干地支表示，共 **4 柱 8 个字**：

```
年柱：天干 + 地支     示例：甲子
月柱：天干 + 地支     示例：丙寅  
日柱：天干 + 地支     示例：戊辰
时柱：天干 + 地支     示例：庚申
```

**日柱天干**称"**日主**"或"**日元**"，是全盘的"我"，其他七个字都以此为参照分析。

## 二、排盘的输入

**必需信息：**
- 出生日期（阳历/公历，YYYY-MM-DD）
- 出生时间（HH:MM，精确到分）
- 出生地点（省市或经度，用于真太阳时修正）
- 性别（男/女，影响大运走向）

## 三、四柱如何排出？

### 1. 年柱

**依据**：以**立春**为界（不是元旦，也不是农历春节）。

- 立春前一秒 → 属于上一年的年柱
- 立春后一秒 → 属于新一年的年柱

**查法**：查万年历，某年立春是几月几日几时几分。

**示例**：2024年立春是2月4日16:27，某人2024年2月4日15:00出生，仍算作**癸卯年**（2023年干支），要到16:27之后才是**甲辰年**。

**天干推算**（可选公式）：
```
年干 = (公历年份 - 3) mod 10  → 对应天干
年支 = (公历年份 - 3) mod 12  → 对应地支
0=甲/子, 1=乙/丑, 2=丙/寅... 
```

### 2. 月柱

**依据**：以**节气**（不是农历月初一）为界，每两个节气对应一个"月"。

**十二月的节气对应**：

| 农历月 | 节气 | 月支 |
|-------|------|------|
| 正月   | 立春 → 惊蛰 | 寅 |
| 二月   | 惊蛰 → 清明 | 卯 |
| 三月   | 清明 → 立夏 | 辰 |
| 四月   | 立夏 → 芒种 | 巳 |
| 五月   | 芒种 → 小暑 | 午 |
| 六月   | 小暑 → 立秋 | 未 |
| 七月   | 立秋 → 白露 | 申 |
| 八月   | 白露 → 寒露 | 酉 |
| 九月   | 寒露 → 立冬 | 戌 |
| 十月   | 立冬 → 大雪 | 亥 |
| 十一月 | 大雪 → 小寒 | 子 |
| 十二月 | 小寒 → 立春 | 丑 |

**月干推算（五虎遁月）**：由年干决定正月起何干。

| 年干 | 正月起 |
|------|-------|
| 甲、己 | 丙寅 |
| 乙、庚 | 戊寅 |
| 丙、辛 | 庚寅 |
| 丁、壬 | 壬寅 |
| 戊、癸 | 甲寅 |

**口诀**："甲己之年丙作首，乙庚之岁戊为头，丙辛必定寻庚起，丁壬壬位顺行流，若言戊癸何方发，甲寅之上好推求。"

### 3. 日柱

**依据**：从任意基准日按六十甲子循环推算。

**基准**：1900年1月1日是"庚戌"日（可用其他基准）。

**算法**：
```
日柱序号 = (总天数差) mod 60
```

或直接查万年历。

**关键**：日柱不受节气/月初影响，是纯粹的天数循环。

### 4. 时柱

**依据**：把一天分为12个时辰，2小时一个。

**时辰对照**（真太阳时）：

| 时辰 | 时间段 | 地支 |
|------|--------|------|
| 子时（早）| 00:00-00:59 | 子 |
| 子时（夜）| 23:00-23:59 | 子（次日）|
| 丑时 | 01:00-02:59 | 丑 |
| 寅时 | 03:00-04:59 | 寅 |
| 卯时 | 05:00-06:59 | 卯 |
| 辰时 | 07:00-08:59 | 辰 |
| 巳时 | 09:00-10:59 | 巳 |
| 午时 | 11:00-12:59 | 午 |
| 未时 | 13:00-14:59 | 未 |
| 申时 | 15:00-16:59 | 申 |
| 酉时 | 17:00-18:59 | 酉 |
| 戌时 | 19:00-20:59 | 戌 |
| 亥时 | 21:00-22:59 | 亥 |

**时干推算（五鼠遁时）**：由日干决定子时起何干。

| 日干 | 子时起 |
|------|-------|
| 甲、己 | 甲子 |
| 乙、庚 | 丙子 |
| 丙、辛 | 戊子 |
| 丁、壬 | 庚子 |
| 戊、癸 | 壬子 |

## 四、时间修正（重要！）

排盘前必须做**真太阳时修正**：

```
真太阳时 = 北京时间 
          - 夏令时（1986-1991年4-9月出生者-1小时）
          + (出生地经度 - 120) × 4 分钟
          + 均时差（可选精修）
```

详见"真太阳时修正说明.txt"。

## 五、大运推算

**大运**是"人生每十年一段的运程"，由月柱推出。

**起大运岁数计算**：
1. 男阳女阴（男命+阳年 or 女命+阴年）→ **顺行**
2. 男阴女阳（男命+阴年 or 女命+阳年）→ **逆行**

**起运岁数**：
- 顺行：从出生日期起，数到下一个节气的天数 ÷ 3 = 起运年龄
- 逆行：从出生日期起，倒数到上一个节气的天数 ÷ 3 = 起运年龄

（3天为1年，1天为4个月，1时辰为10天）

**大运天干地支**：从月柱开始，顺行/逆行推进六十甲子。

**示例**：
- 月柱丙寅，男，阳年（甲子年）出生 → 顺行 → 大运依次为：丁卯、戊辰、己巳、庚午…
- 月柱丙寅，男，阴年（乙丑年）出生 → 逆行 → 大运依次为：乙丑、甲子、癸亥、壬戌…

## 六、排盘输出示例

```
出生：1990年5月20日 上午10:30（北京时间，出生地上海）
性别：男

真太阳时：10:36（经度+6分）

四柱：
       年柱    月柱    日柱    时柱
天干： 庚      辛      辛      癸
地支： 午      巳      未      巳

十神： 劫财   比肩   （日主）  食神

五行统计：金3，水1，木0，火4，土1

大运（顺行，起运年龄 8岁）：
8-17岁：壬午
18-27岁：癸未
28-37岁：甲申
...
```

## 七、断命的核心思路

1. **看日主强弱**：日主五行在整个八字中是强还是弱？
2. **找用神**：需要什么五行来平衡（喜用神）？什么五行是忌讳的（忌神）？
3. **看格局**：正官格、七杀格、财格、印格等
4. **分析十神**：财官印食伤各代表什么，出现在哪一柱
5. **看冲合刑害**：地支之间的相互关系
6. **结合大运流年**：某一年遇什么干支，是吉是凶
