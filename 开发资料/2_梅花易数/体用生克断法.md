# 体用生克断法（梅花易数断卦核心）

## 一、体用的划分

**核心原则**：一个六爻卦分成上下两个三爻卦，其中：

- **体卦**：不含动爻的那个三爻卦（静的一半）→ 代表**求测者本人 / 事情的本体**
- **用卦**：含动爻的那个三爻卦（动的一半）→ 代表**所占之事 / 外部事物**

**记忆诀窍**："静者为体，动者为用"

## 二、动爻位置判断

一个六爻卦，从下往上数是：
- 初爻（第1爻）
- 二爻（第2爻）
- 三爻（第3爻）→ 属于下卦顶部
- 四爻（第4爻）→ 属于上卦底部
- 五爻（第5爻）
- 上爻（第6爻）

**判断动爻在哪个卦**：
- 动爻是1、2、3 → 动在下卦 → **下卦为用，上卦为体**
- 动爻是4、5、6 → 动在上卦 → **上卦为用，下卦为体**

## 三、体用五行生克关系

每个三爻卦有一个五行属性（乾兑金、离火、震巽木、坎水、艮坤土）。

判断体卦与用卦的五行关系：

### 1. 用生体（大吉）

用卦五行生体卦五行 → 外部资源来帮助你 → **吉，得外助**

**举例**：
- 体离（火），用震（木）→ 木生火 → 吉
- 体乾（金），用坤（土）→ 土生金 → 吉

### 2. 用克体（大凶）

用卦克体卦 → 外部力量伤害你 → **凶，事情不利**

**举例**：
- 体离（火），用坎（水）→ 水克火 → 凶
- 体乾（金），用离（火）→ 火克金 → 凶

### 3. 体生用（小凶/漏气）

体卦生用卦 → 自己的能量被外部消耗 → **半吉半凶，付出多**

**举例**：
- 体离（火），用坤（土）→ 火生土 → 泄气，事费力

### 4. 体克用（小吉）

体卦克用卦 → 你能制服外部 → **吉，但耗力**

**举例**：
- 体离（火），用乾（金）→ 火克金 → 你能掌控事情

### 5. 比和（同五行，大吉）

体用五行相同 → **吉，顺利**

**举例**：
- 体震（木），用巽（木）→ 比和 → 顺利

## 五、变卦的判断

动爻变化后形成**变卦**，变卦看**事情的结果**。

**判断方法**：
- 变卦的用卦（即原动爻所在的卦发生变化） → 事物发展的走向
- 变卦仍然生体 → 结果吉
- 变卦克体 → 结果凶

## 六、互卦与旁参

**互卦**：由本卦的2、3、4爻组成下卦，3、4、5爻组成上卦。互卦代表**事情发展过程中的隐性因素**。

**旁参**：结合本卦、互卦、变卦三个卦一起看，能得到更完整的信息。

**综合判断顺序**：
1. 看**体用生克** → 事情整体吉凶
2. 看**变卦** → 结果如何
3. 看**互卦** → 过程中的暗流
4. 结合**卦辞、爻辞** → 具体指引

## 七、断卦例题

**例**：某人问求职是否顺利，起卦得**火风鼎**（第二爻动）

- 本卦：火风鼎（上离下巽）
- 动爻：第2爻（在下卦） → **下卦为用（巽木），上卦为体（离火）**
- 体用关系：用（木）生体（火）→ **大吉！求职可成**
- 变卦：第二爻从阳变阴 → 下卦变成艮（山）→ 火山旅
- 变卦互卦分析：整体走向为"旅"，可能需要外地/换环境
- 结论：**能得到工作，但可能需要异地或流动性较强**

## 八、断卦要点总结

1. **不含动爻的一半是体，代表求测者**
2. **含动爻的一半是用，代表事情**
3. **用生体、体克用、比和 → 吉**
4. **用克体、体生用 → 凶或泄**
5. **变卦看结果，互卦看过程**
6. 结合卦名、卦辞、爻辞的文本意义
