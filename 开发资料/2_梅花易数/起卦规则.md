# 梅花易数起卦规则

## 什么是梅花易数？

宋代邵雍（邵康节）创立的一种简易起卦方法，取材灵活（数字、时间、字数、声音、方位、颜色…），不需要占卜工具。核心思想：**万物都可以起卦**。

传说邵雍见到有人在梅花下折花，起卦推算，故名"梅花易数"。

## 一、起卦的基本流程

1. **取"上卦"**（外卦、动象、事物的整体）
2. **取"下卦"**（内卦、静象、事物的本体）
3. **取"动爻"**（变化的那一爻，是断卦重点）
4. **上卦+下卦 → 本卦（六爻卦）**
5. **动爻变化 → 变卦**（阴变阳/阳变阴）
6. 分析本卦、变卦、体用关系，得出结论

## 二、各种起卦方法

### 方法1：时间起卦（最常用）

**用当前时刻的年月日时**（农历年干支序数 + 月 + 日 + 时辰）。

**公式**：
```
上卦 = (年数 + 月数 + 日数) mod 8    # 余0取8
下卦 = (年数 + 月数 + 日数 + 时数) mod 8    # 余0取8
动爻 = (年数 + 月数 + 日数 + 时数) mod 6    # 余0取6
```

**其中**：
- 年数：地支序数（子=1, 丑=2, 寅=3... 亥=12），或直接用农历年干支的地支
- 月数：农历月份（1-12）
- 日数：农历日（1-30）
- 时数：时辰序数（子=1, 丑=2... 亥=12）

**示例**：
2024年正月初五 巳时起卦
- 甲辰年，辰=5
- 正月=1
- 初五=5
- 巳时=6

上卦 = (5+1+5) mod 8 = 11 mod 8 = 3 → 离 ☲
下卦 = (5+1+5+6) mod 8 = 17 mod 8 = 1 → 乾 ☰
动爻 = (5+1+5+6) mod 6 = 17 mod 6 = 5 → 第五爻动

本卦：离上乾下 → **火天大有**（第五爻动）
变卦：五爻从阴变阳→阳变阴... （需结合火天大有第五爻是阳变阴）→ **乾为天**

### 方法2：数字起卦（问卜时用）

请求卜者报出两个1-10的数字（或直接报一个数字然后取十位/个位）。

**公式**：
```
上卦 = 第一个数 mod 8    # 余0取8
下卦 = 第二个数 mod 8
动爻 = (第一个数 + 第二个数) mod 6    # 余0取6
```

**示例**：报数7和15
- 上卦 = 7 mod 8 = 7 → 艮 ☶
- 下卦 = 15 mod 8 = 7 → 艮 ☶
- 动爻 = (7+15) mod 6 = 4 → 第四爻动

本卦：艮为山（第四爻动）→ 变卦：山火贲

### 方法3：字数起卦

一句话或一个短语，数字数：
- **偶数字数** → 前一半为上卦，后一半为下卦
- **奇数字数** → 前面（多一字）为上卦，后面为下卦
- **总字数** mod 6 → 动爻

**示例**："今日心情不佳"共6字，前3字"今日心"字数=3 → 离，后3字"情不佳"=3 → 离
本卦：重离（离上离下）

### 方法4：声音起卦

- 数听到的声音次数（比如鸟叫3声）
- 或人说话的字数
- 与数字起卦法相同

### 方法5：方位起卦

- 观察问事之人来的方向或物体所在方向
- 按**后天八卦方位**（不是先天）：
  - 北→坎，东北→艮，东→震，东南→巽
  - 南→离，西南→坤，西→兑，西北→乾
- 与其他信息组合（时间等）确定上下卦

### 方法6：物象起卦

- 观察当下环境中的物体（花、云、器物）
- 属性：形状、颜色、五行归属 → 对应卦

## 三、动爻的作用

**动爻是断卦的重点**：

- 动爻处的卦象/爻辞 → 事情的关键
- 动爻变化后 → 事情的结果（变卦）
- 只有 1 个动爻（简易法一般只取一个）

## 四、体用关系（断卦核心）

将六爻卦分成上下两个三爻卦：
- **体卦**：静止的一半（不含动爻的那一卦）→ 代表自己/事主
- **用卦**：变动的一半（含动爻的那一卦）→ 代表所占之事、外部

**判断吉凶**：
- 用生体 → 吉
- 用克体 → 凶
- 体生用 → 泄气（半吉半凶）
- 体克用 → 平（用被制）
- 体用比和（同五行）→ 吉

## 五、断卦步骤总结

1. 起卦得到本卦和变卦
2. 找出动爻位置
3. 分体卦、用卦
4. 分析五行生克关系
5. 结合本卦爻辞、变卦爻辞、卦象象义
6. 综合判断吉凶

## 六、网页实现要点

**起卦方式选择：**
- [x] 时间起卦（推荐默认，无需用户输入）
- [x] 数字起卦（用户输入两个数字）
- [x] 字数起卦（用户输入一句话）

**输出：**
- 本卦（六爻符号 + 卦名 + 卦辞简要）
- 变卦（六爻符号 + 卦名 + 卦辞简要）
- 动爻爻辞
- 体用关系分析
- 吉凶判断（一句话结论）

**算法伪代码：**
```javascript
function meihuaBySteps(upNum, downNum) {
    const upGua = upNum % 8 || 8;    // 上卦
    const downGua = downNum % 8 || 8; // 下卦
    const dongYao = (upNum + downNum) % 6 || 6; // 动爻
    
    const benGua = [upGua, downGua];  // 本卦
    const bianGua = generateBianGua(benGua, dongYao); // 变卦
    
    return { benGua, bianGua, dongYao };
}
```
